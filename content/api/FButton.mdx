# FButton

A tappable button with a text label. Maps to a UIToolkit Button.

## Constructor

### FButton

Creates an FButton element.

| Parameter | Description |
| --- | --- |
| `label` | The text label to display. |
| `onPressed` | The callback invoked on press. Null means no action. |
| `key` | An optional key for reconciliation identity. |

## Properties

### Label

The text label displayed on the button.

### OnPressed

The callback invoked when the button is pressed. Null means the button is present but performs no action.

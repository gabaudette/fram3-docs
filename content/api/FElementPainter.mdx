# FElementPainter

Responsible for creating and updating native UIToolkit VisualElement instances to match a given FElement description.

## Methods

### ApplyAsStackChild

Applies absolute positioning to a native element so that it behaves as a layer inside an FStack container. Called by the renderer after attaching the child to a stack parent.

### CreateNative

Creates the appropriate native VisualElement for the given element and applies all initial style properties to it.

| Parameter | Description |
| --- | --- |
| `element` | The framework element to produce a native element for. |

**Returns:** A Label for FText, a Button for FButton, a TextField for FTextField, a Toggle for FToggle, a Slider for FSlider, a DropdownField for FDropdown, a ProgressBar for FProgressBar, a ScrollView for FScrollView, an Image for FImage or FIcon, an FSpinnerElement for FSpinner, or a plain VisualElement for all layout/container/gesture elements.

### Paint

Updates an existing native VisualElement to reflect the latest property values from the given element description.

| Parameter | Description |
| --- | --- |
| `element` | The current element description. |
| `native` | The existing native element to update. |

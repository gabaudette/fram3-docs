# PlayerPrefsAdapter

An IPersistenceAdapter that persists state as JSON in UnityEngine.PlayerPrefs. Suitable for lightweight state that fits comfortably in a string (settings, preferences, small session data).

## Constructor

### PlayerPrefsAdapter

Creates an adapter that stores state under the given PlayerPrefs key.

| Parameter | Description |
| --- | --- |
| `key` | The PlayerPrefs key under which the state JSON is stored. |

# FWrap

Lays out its children in a horizontal sequence that wraps onto the next line when there is not enough space. Useful for tag clouds, item grids, and ability bars. Maps to a UIToolkit VisualElement with FlexDirection.Row and FlexWrap.Wrap.

## Constructor

### FWrap

Creates an FWrap element.

| Parameter | Description |
| --- | --- |
| `key` | An optional key for reconciliation identity. |

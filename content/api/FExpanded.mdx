# FExpanded

Wraps a single child and causes it to expand along the main axis of the parent flex container, proportional to its Flex factor. Maps to UIToolkit's flexGrow style property.

## Constructor

### FExpanded

Creates an FExpanded element.

| Parameter | Description |
| --- | --- |
| `flex` | The flex grow factor. Must be greater than zero. Defaults to 1. |
| `key` | An optional key for reconciliation identity. |

## Properties

### Flex

The flex factor that controls how much of the remaining space this element claims relative to other FExpanded siblings. Defaults to 1.

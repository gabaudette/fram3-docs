# FProvider

Makes a value of type available to any descendant in the element tree. Descendants retrieve it with context.GetInherited&lt;FProvider&lt;T&gt;&gt;().Value or by using FConsumer / FSelector.

| Parameter | Description |
| --- | --- |
| `value` | The value to provide. May be null for reference types. |
| `child` | The child subtree that can access this provider. Must not be null. |
| `key` | An optional key for reconciliation identity. |

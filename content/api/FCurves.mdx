# FCurves

Provides a set of pre-built easing curves for use with FAnimationController and FAnimationBuilder.

## Fields

### BounceOut

Bounces at the end, simulating a ball dropping onto a surface.

### EaseIn

Starts slowly and accelerates toward the end (quadratic ease-in).

### EaseInOut

Accelerates in the first half and decelerates in the second half (cubic ease-in-out).

### EaseOut

Starts quickly and decelerates toward the end (quadratic ease-out).

### ElasticOut

Overshoots the target slightly before settling, producing a spring-like feel. Uses a cubic back ease-out.

### Linear

Constant linear progression. Output equals input.

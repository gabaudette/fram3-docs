# FPadding

Insets its single child by the given FEdgeInsets. Maps to padding applied directly to the child's containing VisualElement.

## Constructor

### FPadding

Creates an FPadding element.

| Parameter | Description |
| --- | --- |
| `padding` | The padding insets to apply. |
| `key` | An optional key for reconciliation identity. |

## Properties

### Padding

The amount of padding to apply on each edge.

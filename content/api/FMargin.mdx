# FMargin

Applies an outer margin to its single child by the given FEdgeInsets. Maps to margin applied directly to the child's containing VisualElement.

## Constructor

### FMargin

Creates an FMargin element.

| Parameter | Description |
| --- | --- |
| `margin` | The margin insets to apply. |
| `child` | The child element to apply margin to. Must not be null. |
| `key` | An optional key for reconciliation identity. |

## Properties

### Margin

The amount of margin to apply on each edge.

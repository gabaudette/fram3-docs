# FRouteElement

Internal stateless element that evaluates a route builder function. Created by FNavigatorState each time a route is pushed or the initial route is mounted. Not intended for direct use.

## Properties

### Arguments

The optional arguments that were passed to Object). Available via context.GetInherited<FNavigatorScope>().Navigator but more commonly passed through application-level state.

# Border

Describes a uniform border with a color and width applied to all four sides of an element.

## Constructor

### Border

Describes a uniform border with a color and width applied to all four sides of an element.

| Parameter | Description |
| --- | --- |
| `Color` | The color of the border. |
| `Width` | The width of the border in logical pixels. |

## Properties

### Color

The color of the border.

### None

A border with zero width and transparent color.

### Width

The width of the border in logical pixels.

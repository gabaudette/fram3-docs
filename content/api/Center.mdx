# Center

Centers its single child both horizontally and vertically within the available space. Maps to a UIToolkit VisualElement with alignItems: center and justifyContent: center.

## Constructor

### Center

Creates an Center element.

| Parameter | Description |
| --- | --- |
| `key` | An optional key for reconciliation identity. |

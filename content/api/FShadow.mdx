# FShadow

Describes a drop shadow cast by an element.

## Constructor

### FShadow

Describes a drop shadow cast by an element.

| Parameter | Description |
| --- | --- |
| `Color` | The color of the shadow. |
| `OffsetX` | Horizontal offset of the shadow in logical pixels. Positive values move the shadow right. |
| `OffsetY` | Vertical offset of the shadow in logical pixels. Positive values move the shadow down. |
| `BlurRadius` | The blur radius in logical pixels. A value of 0 produces a sharp shadow. |

## Properties

### BlurRadius

The blur radius in logical pixels. A value of 0 produces a sharp shadow.

### Color

The color of the shadow.

### OffsetX

Horizontal offset of the shadow in logical pixels. Positive values move the shadow right.

### OffsetY

Vertical offset of the shadow in logical pixels. Positive values move the shadow down.

## Methods

### Ambient

Creates a shadow with no offset and no blur.

| Parameter | Description |
| --- | --- |
| `color` | The shadow color. |
| `blur` | The blur radius. |

**Returns:** A centered FShadow with the given color and blur.

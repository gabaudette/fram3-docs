# Padding

Insets its single child by the given EdgeInsets. Maps to padding applied directly to the child's containing VisualElement.

## Constructor

### Padding

Creates an Padding element.

| Parameter | Description |
| --- | --- |
| `padding` | The padding insets to apply. |
| `key` | An optional key for reconciliation identity. |

## Properties

### Insets

The amount of padding to apply on each edge.

# FThemeProvider

Makes an FTheme available to all descendants. Descendants retrieve the current theme via FBuildContext). When the parent rebuilds with a different FTheme instance, all dependents are automatically scheduled for a rebuild.

## Constructor

### FThemeProvider

Creates an FThemeProvider that supplies to the subtree rooted at .

| Parameter | Description |
| --- | --- |
| `theme` | The theme to provide. Must not be null. |
| `child` | The child subtree that can access the theme. Must not be null. |
| `key` | An optional key for reconciliation identity. |

## Properties

### Theme

The theme made available to descendants.

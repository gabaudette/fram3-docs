# FBoxDecoration

Describes the visual decoration of a box, including background color, border, corner radii, and shadow. Used with container elements such as FContainer.

## Constructor

### FBoxDecoration

Describes the visual decoration of a box, including background color, border, corner radii, and shadow. Used with container elements such as FContainer.

| Parameter | Description |
| --- | --- |
| `Color` | The background fill color. Null means no fill. |
| `Border` | The border drawn around the box. Null means no border. |
| `BorderRadius` | The corner radii of the box. Null means sharp corners. |
| `Shadow` | A drop shadow cast by the box. Null means no shadow. |

## Properties

### Border

The border drawn around the box. Null means no border.

### BorderRadius

The corner radii of the box. Null means sharp corners.

### Color

The background fill color. Null means no fill.

### None

A decoration with no fill, no border, no radius, and no shadow.

### Shadow

A drop shadow cast by the box. Null means no shadow.

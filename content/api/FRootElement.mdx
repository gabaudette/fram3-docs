# FRootElement

A synthetic single-child element that serves as the invisible root of every framework tree. FRenderer wraps the user-supplied root element inside an FRootElement so the reconciler always has a stable anchor node at depth zero whose native counterpart is the UIToolkit container passed to VisualElement).

# Divider

A thin horizontal or vertical separator line with no children. Maps to a UIToolkit VisualElement with a fixed thickness, a background color, and an optional margin.

## Constructor

### Divider

Creates an Divider element.

| Parameter | Description |
| --- | --- |
| `axis` | The draw axis. Defaults to Horizontal. |
| `thickness` | Line thickness in logical pixels. Defaults to 1. |
| `color` | Line color. Null means no explicit color is applied. |
| `indent` | Perpendicular indent in logical pixels. Defaults to 0. |
| `key` | An optional key for reconciliation identity. |

## Properties

### Axis

The axis along which the divider is drawn.

### Color

The color of the divider line. Null uses the inherited foreground color.

### Indent

Space added on each side perpendicular to the divider axis. Defaults to 0.

### Thickness

The thickness of the divider line in logical pixels. Defaults to 1.

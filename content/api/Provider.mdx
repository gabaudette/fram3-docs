# Provider

Makes a value of type available to any descendant in the element tree. Descendants retrieve it with context.GetInherited&lt;Provider&lt;T&gt;&gt;().Value or by using Consumer / Selector.

| --- | --- |

| `child` | The child subtree that can access this provider. Must not be null. |
| `key` | An optional key for reconciliation identity. |

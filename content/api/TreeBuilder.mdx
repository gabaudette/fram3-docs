# TreeBuilder

Provides factory methods for assembling small element trees and expander/scheduler pairs used across integration test suites.

## Methods

### Flush

Flushes all pending rebuilds through .

### MakePipeline

Creates a fresh FRebuildScheduler and FNodeExpander wired to the given adapter. Both objects are ready to use immediately.

### Mount

Mounts and returns the root FNode.
